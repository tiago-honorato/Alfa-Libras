# Caça Sinais (Alfa Libras) 🤟📚  
**Caça Sinais (Alfa Libras)** É um aplicativo multiplataforma (Mobile, Web e Windows) adaptado para o ensino e prática da Língua Brasileira de Sinais (LIBRAS).

🎮 **Descrição do Jogo**

Projeto de desenvolvimento focado na criação de um aplicativo estilo caça-palavras, adaptado para o ensino e prática da Língua Brasileira de Sinais (LIBRAS). O objetivo principal é associar o entretenimento à educação para surdos e ouvintes, utilizando o alfabeto datilológico e configurações de mão da LIBRAS em substituição às letras do alfabeto latino tradicional. O desenvolvimento envolve etapas de design de interface, desenvolvimento de software via Unity Engine e validação pedagógica do conteúdo.

## 📌 **Requisitos do Sistema**
### Windows
- **SO**: Requer sistema operacional e processador de 64 bits  
- **Processador**: Arquitetura x64
- **Memória**: 1 GB de RAM  
- **Placa de Vídeo**: Compatível com DX11 ou DX12  
- **Armazenamento**: 250 MB de espaço disponível

## 🛠️ **Desenvolvimento**
O **Caça Sinais (Alfa Libras)** foi desenvolvido utilizando a Unity Engine. O projeto reflete o esforço de criar uma experiência acessível, lúdica e enriquecedora para todos os jogadores, unindo mecânicas clássicas de jogos digitais à tecnologia assistiva e inclusão sociolinguística.

### Tecnologias Utilizadas
- Unity Engine
- Linguagem C#
- Cloudinary
